# OVA2QCOW2

Convert VirtualBox OVA format to qcow2.

## Dependencies

- qemu-img
- POSIX tar

## Help

ova2qcow2

    Usage: ova2qcow2 OVA QCOW
    
    Convert VirtualBox OVA format to qcow2.
